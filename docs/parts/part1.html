AND102 Activities and Intents

Exercise 1: Explore the completed exercise

This exercise asks you to run an app that demonstrates the course goals. The image below shows the various screens of the app. The lines indicate the transitions between Activities and the arrowheads show the direction of data flow. Your goal in this exercise is to work with the app so you understand its structure - subsequent exercises will ask you to begin with a starter version of this app and write the code to implement all the Activity transitions.

To complete the exercise, you will need Visual Studio for Windows or macOS with the Xamarin development tools installed. You will also need either an emulator/simulator or a device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Open the completed solution

Open your copy of the cloned or downloaded Exercise 1 > Completed solution from your copy of the cloned or downloaded course materials in either Visual Studio for Windows or Visual Studio for Mac.

Run the application

  1. Build and run the Grocery List application.

Navigate to and from an Activity

Below are the step-by-step instructions to navigate between Activities.

  1. Touch the ITEMS button to navigate to the Items Activity.
  1. Touch one of the items in the list to navigate to the Details Activity. The Items Activity will pass the item's position in the data model to the Details Activity so it knows which item to display.
  1. Touch the Android back-button to return to the Items Activity.
  1. Touch the Android back-button to return to the Main Activity.

Add data to the application

Below are the step-by-step instructions to add data to the application.

  1. Touch the ADD ITEM button to navigate to the Add Item Activity. Enter values for a new item.
  1. Touch the SAVE button. The Add Item Activity will send the values for the new item back to the Main Activity.
  1. Touch the ITEMS button to move to the Items Activity. Verify that the new item was added to the list.
  1. Touch the Android back-button to return to the Main Activity.

Launch an external Activity

Below are the step-by-step instructions to launch an external Activity.

  1. Touch the ABOUT button to navigate to the About Activity.
  1. Touch the LEARN MORE button to launch the web browser. The web browser is a standard Android Activity that your app is using.

Exercise summary

In this exercise, you ran an app that let you navigate between Activities, add data, and launch an external activity. Now that you know what the completed application will look like, let's get started building it.
