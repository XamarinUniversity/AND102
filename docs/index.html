Activities and Intents (AND102)

Most apps contain multiple pages. The most common way to implement this in Android is using one Activity for each page and stack navigation to transition between pages. This course shows you how to code the core patterns that make Activities work together. You will see how to start Activities, pass arguments, and retrieve results. You will also see how to launch standard Android Activities such as the camera or the web browser.

Recommended prerequisite for this course

  • Introduction to Xamarin.Android (AND101)

Upon completion you will be able to:

  • Start an Activity in your .apk
  • Finish an Activity
  • Pass arguments to an Activity
  • Get Activity results
  • Launch a system Activity

Exercises

  1. Exercise 1: Explore the completed exercise

    This exercise asks you to run an app that demonstrates the course goals. The image below shows the various screens of the app. The lines indicate the transitions between Activities and the arrowheads show the direction of data flow. Your goal in this exercise is to work with the app so you understand its structure - subsequent exercises will ask you to begin with a starter version of this app and write the code to implement all the Activity transitions.

  2. Exercise 2: Start an Activity in your .apk

    The goal of this exercise is to use Explicit Intents to start Activities from your application package.

  3. Exercise 3: Programmatically end an Activity

    The goal of this exercise is to demonstrate how to end an Activity programmatically. This is useful whenever you need to implement something like a Save or Cancel button in your user interface.

  4. Exercise 4: Pass arguments to an activity

    The goal of this exercise is to pass arguments to an Activity. You will see how to load arguments in the source Activity and how to retrieve them in the target Activity.

  5. Exercise 5: Get activity results

    The goal of this exercise is to return results from an Activity. You will see how to load the results in the target Activity and how to retrieve them in the source Activity.

  6. Exercise 6: Launch a system activity

    The goal of this lab is to demonstrate how to launch an Android system Activity (i.e. an Activity installed with Android and not part of your application).
